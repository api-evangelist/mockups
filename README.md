# Mockups (mockups)

Mockups are visual representations or prototypes of user interfaces and designs used to demonstrate the look and feel of a product before development. They are used across software, mobile, web, and product design to align stakeholders, validate ideas, and improve outcomes by surfacing feedback before code is written. This index tracks tools, services, and resources in the mockup and prototyping space.
